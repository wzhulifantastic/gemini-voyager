# Masquer automatiquement la barre latérale

Vous voulez une expérience de chat plus immersive ?

Nous proposons une fonctionnalité de **Masquage automatique de la barre latérale**. Lorsqu'elle est activée, la barre latérale se réduit automatiquement lorsque votre souris quitte la zone, et se développe automatiquement lorsque vous y revenez.

### Comment activer

1. Ouvrez le panneau de configuration de Gemini Voyager.
2. Trouvez l'option **Masquer auto barre latérale** dans les **Paramètres généraux**.
3. activez l'interrupteur.

_Remarque : Cette fonctionnalité ne prend actuellement en charge que Google Gemini._
