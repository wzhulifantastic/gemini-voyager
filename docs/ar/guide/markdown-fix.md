# إصلاح عرض Markdown

تقوم واجهة ويب Gemini أحياناً بإدراج عناصر HTML (مثل مصادر الاقتباس أو علامات التمييز) داخل النص، مما قد يؤدي إلى كسر تنسيق Markdown للنص العريض (`**النص**`)، ويتسبب في عدم ظهور النص بشكل عريض صحيحاً.

يتضمن Gemini Voyager ميزة إصلاح تلقائي مدمجة تتعرف بذكاء على علامات النص العريض المكسورة هذه وتقوم بإصلاحها، مما يضمن عرض مستنداتك بشكل نظيف ودقيق.

> [!INFO]
> هذه الميزة مفعلة تلقائياً ولا تتطلب أي إعدادات إضافية.
