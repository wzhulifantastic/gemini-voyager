# 입력창 접기

입력창이 비어 있을 때 이를 접어서 더 많은 읽기 공간을 확보하세요. 접힌 바를 클릭하면 다시 펼쳐져서 타이핑을 시작할 수 있습니다.

</div>

## 사용 방법

1. 입력창이 비어 있고 포커스를 잃으면, 자동으로 콤팩트한 알약 모양의 버튼으로 접힙니다.
2. 알약 버튼을 클릭하면 입력창이 펼쳐지며 타이핑을 시작할 수 있습니다.
3. 설정 패널에서 이 기능을 켜거나 끌 수 있습니다 (기본적으로 비활성화되어 있습니다).
