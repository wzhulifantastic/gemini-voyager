# Синхронизация контекста: Передача воспоминаний (Экспериментально)

**Разные измерения, бесшовный обмен**

Прорабатывайте логику в вебе, а код реализуйте в IDE. Gemini Voyager разрушает барьеры между измерениями, мгновенно передавая вашему IDE «мыслительный процесс» из веба.

## Попрощайтесь с постоянными переключениями

Главная боль разработчиков: после детального обсуждения решения в вебе вы возвращаетесь в VS Code/Trae/Cursor, только чтобы заново объяснять требования, как незнакомцу. Из-за квот и скорости ответа веб — это «мозг», а IDE — «руки». Voyager позволяет им разделить одну душу.

## Три простых шага для синхронизации

1. **Запустите CoBridge**: Установите расширение **CoBridge** из VS Code Marketplace и запустите его. Это мост, соединяющий веб и вашу локальную машину.
   ![Расширение CoBridge](/assets/CoBridge-extension.png)

   ![Сервер CoBridge включен](/assets/CoBridge-on.png)

2. **Установка соединения**:
   - Включите «Context Sync» (Синхронизация контекста) в настройках Voyager.
   - Согласуйте номера портов. Когда вы увидите «IDE Online», они подключены.

   ![Консоль синхронизации контекста](/assets/context-sync-console.png)

3. **Синхронизация в один клик**: Нажмите **«Sync to IDE»**.

   ![Синхронизация завершена](/assets/sync-done.png)

## Укоренение в IDE

После синхронизации в корневом каталоге вашей IDE появится файл `.vscode/AI_CONTEXT_SYNC.md`. Будь то Trae, Cursor или Copilot, они автоматически считают эту «память» через свои соответствующие файлы правил. **ИИ-модели больше не будут страдать от потери памяти, сразу включаясь в работу.**

## Принципы

- **Никакого загрязнения**: CoBridge автоматически обрабатывает `.gitignore`, гарантируя, что ваши личные разговоры никогда не попадут в репозитории Git.
- **Промышленный стандарт**: Полный формат Markdown, что делает чтение для ИИ в вашей IDE таким же простым, как чтение инструкции.
- **Совет профи**: Если разговор был давно, сначала прокрутите вверх, используя [Timeline] (Хронологию), чтобы веб «вспомнил» контекст для лучших результатов синхронизации.
